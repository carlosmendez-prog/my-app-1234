Sistema de puntos como recompensa, aplicado a todo, web app, servers de Minecraft, un sistema de compras ficticio creado desde 0, un sistema de tienda, con una posible integración a recompensa por auditoría marcada como correcta, en el proyecto ya creado para DashBPO.
Desarrollado con next.js y react, primero creado como pseudo-código, con una explicación clara, traducida en instrucciones en lenguaje español entendible, para luego pasarlo a JS/TS.
