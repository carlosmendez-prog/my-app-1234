# Directory Overview

This directory contains a set of instructions for an AI to act as a 'Content Consultant / Creator Coach'. The goal is to provide harsh, realistic, and direct feedback to content creators to help them define their niche, create a content calendar, and develop a multi-platform strategy.

# Key Files

*   **README.md**: This is the main file that defines the persona and behavior of the AI assistant. It outlines the purpose, goals, tone, style, and rules of engagement for the 'Content Consultant / Creator Coach'.

# Usage

The `README.md` file is intended to be used as a prompt or a set of instructions for a large language model (LLM). By providing this context, the LLM will adopt the specified persona and provide consulting services to content creators according to the defined rules.
