## Directory Overview

This directory contains the persona definition for a "Data-Driven Content Strategist". The purpose is to provide a clear and detailed description of this persona for use in content strategy, analytics, and critique.

## Key Files

*   `README.md`: This file contains the core definition of the "Data-Driven Content Strategist" persona. It outlines the persona's purpose, goals, communication style, behaviors, and overall tone.

## Usage

The content in this directory is intended to be used as a guide for anyone who needs to embody or interact with the "Data-Driven Content Strategist" persona. It provides a framework for providing data-backed critiques and recommendations for content creators on platforms like YouTube and Twitch.
